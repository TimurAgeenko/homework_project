# Проект "Виджет"

## Описание:

Проект "Виджет" - это виджет в приложении банка, который показывает несколько последних успешных банковских операций клиента.

## Установка:

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/TimurAgeenko/homework_project
   ```
2. Установите зависимости:
   ```
   poetry install
   ```

## Использование:

1. Функцию get_mask_card_number можно использовать для создания замаскированного варианта номера карты:
   ```
   get_mask_card_number("7000792289606361")
   
   Выдаст "7000 79** **** 6361"
   ```
2. Функцию get_mask_account можно использовать для создания замаскированного варианта номера счета:
   ```
   get_mask_account("73654108430135874305")
   
   Выдаст "**4305"
   ```
3. Функция mask_account_card принимает тип и номер карты или счета и возвращает замаскированный вариант:
   ```
   mask_account_card("Visa Platinum 7000792289606361")
   
   Выдаст "Visa Platinum 7000 79** **** 6361"
   
   
   mask_account_card("Счет 73654108430135874305")
   
   Выдаст "Счет **4305"
   ```
4. Функция get_date принимает строку с датой в одном формате и возвращает в другом:
   ```
   get_date("2024-03-11T02:26:18.671407")
   
   Выдаст "11.03.2024"
   ```
5. Функция filter_by_state принимает список с информацией и значение state, по умолчанию равное 'EXECUTED' и возвращает отредактированный список
   со словарями, у которых ключ state принимает указанное значение:
   ```
   filter_by_state([
   {'id': 41428829, 'state': 'EXECUTED'},
   {'id': 939719570, 'state': 'EXECUTED'},
   {'id': 594226727, 'state': 'CANCELED'},
   {'id': 615064591, 'state': 'CANCELED'}
   ])
   
   Выдаст [
   {'id': 41428829, 'state': 'EXECUTED'},
   {'id': 939719570, 'state': 'EXECUTED'}
   ]
   ```
6. Функция sort_by_date принимает список и возвращает его отсортированный по дате вариант, по умолчанию - по убыванию:
   ```
   sort_by_date([
   {'id': 41428829, 'date': '2019-07-03T18:35:29.512364'},
   {'id': 939719570, 'date': '2018-06-30T02:08:58.425572'},
   {'id': 594226727, 'date': '2018-09-12T21:27:25.241689'},
   {'id': 615064591, 'date': '2018-10-14T08:21:33.419441'}
   ])
   
   Выдаст [
   {'id': 41428829, 'date': '2019-07-03T18:35:29.512364'},
   {'id': 615064591, 'date': '2018-10-14T08:21:33.419441'},
   {'id': 594226727, 'date': '2018-09-12T21:27:25.241689'},
   {'id': 939719570, 'date': '2018-06-30T02:08:58.425572'}
   ]
   ```

## Тестирование

Для запуска тестов выполните следующие шаги:

1. Убедитесь, что у вас установлен `pytest`
2. В терминале наберите `pytest`, чтобы запустить тестирование
3. Проверьте результаты в консоли

При тестировании используются юнит-тесты, их результаты выводятся в консоль.

Подробные отчеты о тестировании можно найти в директории `/htmlcov/`
